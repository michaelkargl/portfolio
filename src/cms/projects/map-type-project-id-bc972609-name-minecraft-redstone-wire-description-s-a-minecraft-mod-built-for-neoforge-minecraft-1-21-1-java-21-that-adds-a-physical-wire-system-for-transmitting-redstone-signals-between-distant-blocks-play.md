name: "Minecraft: Redstone Wire"
description: A Minecraft mod built for NeoForge (Minecraft 1.21.1, Java 21) that
  adds a physical wire system for transmitting redstone signals between distant
  blocks. Players place specialized Input, Output, and Connector blocks, then
  link them together using a two-click workflow with redstone dust. Connector
  blocks use a lazy DFS cache to discover all reachable outputs, rebuilding only
  on topology changes to keep the hot path (up to 20 ticks/sec) efficient
