name: PlatformIO GDLib HAGL HAL
description: A PlatformIO-compatible Hardware Abstraction Layer (HAL) for the
  HAGL graphics library that implements software rasterization entirely in C.
  Instead of a display, it maps a pixel buffer to ASCII characters and renders
  to stdout, making it useful for testing graphics code without hardware.
  Includes a Mandelbrot set renderer and text rendering examples.
